This directory contains the scripts used to rarefy and plot mtDNA haplotypes

Rarefaction_curve.R -- script that uses rarefaction to estimate mtDNA richness for each population

Boxplot_mtDNA_richness.R -- script for creating boxplots of mtDNA richness by 

Plot_mtDNA_richness_by_geography.R -- script used for plotting haplotype richness by subpopulation on global map

Plotting_COI_haplotypes_by_geography.R -- script used for plotting the haplotype of each individual on global map
